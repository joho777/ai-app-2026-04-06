# ShipSprint AI

A single-file interactive web app inspired by the trend: **"Eight years of wanting, three months of building with AI"**.

ShipSprint AI helps you compress a vague idea into:
- A compact spec with milestones, acceptance criteria, and edge cases
- A copy/paste prompt pack for product, UI, and QA
- A mini sprint simulation (time, quality, scope, stress tradeoffs)
- A crisp demo pitch (plus a tweet draft)

## How to use

1. Open `index.html` in any modern browser.
2. Enter your idea, audience, and shipping window.
3. Toggle constraints/ingredients and choose an archetype.
4. Click **Generate Plan** (or press **Ctrl + Enter**).
5. Run the Sprint Sim and export artifacts.

## Notes

- Local-only: no network calls.
- Everything is embedded in `index.html` (CSS + JS inline).
